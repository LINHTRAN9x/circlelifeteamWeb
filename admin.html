🗄️ Cấu Hình JSONBin.io

⚠️ Cấu hình trong file js/config.js

Hướng dẫn thiết lập JSONBin.io:

  1. Truy cập jsonbin.io và tạo tài khoản miễn phí
  2. Vào mục API Keys → tạo Master Key mới
  3. Vào Bins → Create New Bin
  4. Paste dữ liệu mẫu bên dưới vào bin và lưu
  5. Copy Bin ID và API Key
  6. Mở file js/config.js và thay thế các giá trị

          📖
          Hướng Dẫn Sử Dụng
        


        

          

            
1️⃣

            
Thêm Game Mới

            
Vào trang Quản Lý Game → Bấm Thêm Game Mới → Điền thông tin → Lưu

          

          

            
2️⃣

            
Cập Nhật Game

            
Bấm nút ✏️ Sửa bên cạnh tên game → Chỉnh sửa thông tin → Lưu lại

          

          

            
3️⃣

            
YouTube Trailer

            
Chỉ điền ID video YouTube (phần sau ?v=), không cần full URL

          

          

            
4️⃣

            
Ảnh Màn Hình

            
Điền mỗi URL ảnh trên một dòng trong ô "Ảnh màn hình"

          

          

            
5️⃣

            
Đổi Mật Khẩu

            
Tạo MD5 hash cho mật khẩu mới tại md5hashgenerator.com → Paste vào config.js

          

          

            
6️⃣

            
SEO & URL

            
URL game tự tạo từ Slug. Ví dụ: game.html?id=ghost-of-tsushima
